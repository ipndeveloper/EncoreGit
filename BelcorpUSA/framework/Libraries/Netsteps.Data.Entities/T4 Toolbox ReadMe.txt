In order to get the T4 templates to work, download and install the T4 Toolbox from 

http://t4toolbox.codeplex.com/

and add an environment variable pointing to the t4toolbox folder in Program Files(x86).

Visual Studio > Tools, > Options, > Environment, > Add-in/Macros Security, > Add-in File Paths

Add the path "C:\Program Files(x86)\T4 Toolbox\"

Restart Visual Studio.
